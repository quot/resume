[resume.acote.dev](https://resume.acote.dev/)

## Requirements

- `make`
- `pandoc`
- `node`
- `weasyprint`

## Build

Full:

Set `RESUME_EMAIL` and `RESUME_PHONE` in your shell or CI environment, then run:

```sh
make
```

Web Only:

Set `RESUME_EMAIL` and `RESUME_PHONE` in your shell or CI environment, then run:

```sh
make web
```
